Easily build forms for your MailChimp lists, add them to posts, pages & sidebars and track subscriber activity.== Description == Easy Forms for MailChimp allows you to add MailChimp sign up forms to your WordPress site. You can add forms to posts, pages, sidebars and other widgetized areas. Your MailChimp API Key connects your site to your MailChimp account and pulls in all of your MailChimp list information. List statistics and user activity can be viewed right from your dashboard. From there, you can manage users and list settings.**NOTE:** Version 6 is a major rewrite of the plugin and a few things have changed, [please visit our Knowledge Base](https://yikesinc.freshdesk.com/support/solutions/folders/6000128806) for articles on upgrading to version 6.You can also extend the plugin's functionality with our [free and paid add-ons](https://yikesplugins.com/?utm_source=wp_plugin_repo&utm_medium=link&utm_campaign=easy_forms_for_mailchimp).**Features*** Connect your website to your MailChimp account with a MailChimp forms.* Import/Export all MailChimp forms and associated settings for use on other sites with Easy Forms for MailChimp installed.* Fully customize the success and error messages displayed back to Download the plugin .zip file and make note of where on your computer you downloaded it to.2. In the WordPress admin (yourdomain.com/wp-admin) go to Plugins > Add New or click the "Add New" button on the main plugins screen.3. On the following screen, click the "Upload Plugin" button.4. Browse your computer to where you downloaded the plugin .zip file, select it and click the "Install Now" button.5. After the plugin has successfully installed, click "Activate Plugin" and enjoy!Visit [our Knowledge Base](https://yikesinc.freshdesk.com/support/solutions/6000083118) for more detailed installation instructions.== Frequently Asked Questions ==All documentation can be found in [our Knowledge Base](https://yikesinc.freshdesk.com/support/solutions/6000083118). = Do I need a MailChimp account to use this plugin? =Yes, you must have a MailChimp account and at least 1 list set up in order to use the plugin. You can register for one for free at [MailChimp](https://login.mailchimp.com/signup? "MailChimp").= How does the plugin connect my MailChimp account to my site? =Our plugin uses a MailChimp API Key to allow your site to communicate with your MailChimp account. After you enter your API key into the plugin settings your site is connected and ready to make forms.= How do I create a MailChimp API key? =Go to the MailChimp article [About API Keys](https://kb.mailchimp.com/accounts/management/about-api-keys) for detailed information on creating API keys.Step-by-step instructions from MailChimp:1. Click your profile name to expand the Account Panel, and choose Account.2. Click the Extras drop-down menu and choose API keys.3. Copy an existing API key or click the Create A Key button.4. Name your key descriptively, so you know what application uses that key.Copy your key so you can paste it into the plugin General Settings.= I upgraded the plugin to version 6 and my forms disappeared! What's going on? =Version 6 of the plugin was a complete rewrite of the codebase to make the plugin better than ever. In order to do this we needed to update the shortcode format. Simply replace the old shortcodes with the new one and your forms will be back. For more information on shortcodes visit our [Knowledge Base](https://yikesinc.freshdesk.com/support/solutions/articles/6000056637-using-shortcodes).= I'm having trouble upgrading to version 6, what do I do? ==We want our current users to have as smooth a transition as possible so we have created [Guides to Updating](https://yikesinc.freshdesk.com/support/solutions/folders/6000128806). Please check them out if you are experiencing problems.= I upgraded the plugin to version 6 and now Custom Form Styles and Templates are gone. What's up with that? =In order to sustain the development of this plugin, and make it the best quality possible, we needed to move form customization to a [paid add-on](https://yikesplugins.com/plugin/form-customizer-for-easy-forms-for-mailchimp/?utm_source=wp_plugin_repo&utm_medium=link&utm_campaign=easy_forms_for_mailchimp). The Form Customizer add-on is a powerful and easy-to-use tool to customize the look and feel of your forms without needing to know any code. If you want ultimate control of the style of your forms, please [give this add-on a try](https://yikesplugins.com/plugin/form-customizer-for-easy-forms-for-mailchimp/?utm_source=wp_plugin_repo&utm_medium=link&utm_campaign=easy_forms_for_mailchimp).== Screenshots ==1.
